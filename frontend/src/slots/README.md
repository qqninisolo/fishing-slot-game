## 核心功能说明

### 1. 通用基类 SlotMachine
所有Slots主题继承此类，实现了以下通用功能：
- ✅ 5轴3行20条支付线标准结构
- ✅ 随机转轮生成逻辑
- ✅ 中奖线判定算法
- ✅ 奖励计算逻辑
- ✅ 免费旋转、乘數机制
- ✅ 自动旋转支持
- ✅ 下注金额校验

### 2. 3款主题Slots特性

#### 🌊 海洋主题 OceanSlot
- **特色**：与捕鱼游戏资产互通，捕鱼获得的特殊道具可在此使用
- **特殊符号**：
  - Wild（海马）：替代除Scatter外所有符号
  - Scatter（贝壳）：3个以上触发15次免费旋转，旋转期间奖励翻倍
  - Bonus（宝箱）：3个以上触发捕鱼小游戏，奖励直接进入捕鱼账户
- **RTP**：94%

#### 🏛️ 古埃及主题 EgyptSlot
- **特色**：高风险高回报，最高奖励可达10000倍下注
- **特殊符号**：
  - Wild（法老）：叠加倍率，每个Wild翻倍，最高x5
  - Scatter（金字塔）：3个以上触发免费旋转，最多可叠加到100次免费旋转
  - Bonus（木乃伊）：3个以上触发寻宝小游戏，最高奖励10000倍
- **RTP**：92%

#### 💰 财富宝藏主题 TreasureSlot
- **特色**：低门槛高中奖率，适合新手用户
- **特殊符号**：
  - Wild（金币）：出现即奖励对应倍数，每个奖励2倍
  - Scatter（元宝）：2个即可触发免费旋转
  - Bonus（财神）：3个以上触发红包雨，最低奖励10倍下注
- **RTP**：96%
- **特有规则**：2个相同符号即可中奖，小奖多，体验友好

## 快速开始

### 基础使用
```javascript
// 导入对应主题
import OceanSlot from './themes/ocean/OceanSlot';

// 初始化
const slot = new OceanSlot();

// 设置下注
slot.setBet(100);

// 旋转
const result = slot.spin();

// 结果结构
console.log(result);
// {
//   reels: [[...], [...], [...], [...], [...]], // 5轴3行符号矩阵
//   wins: [...], // 中奖列表，包含每条线的奖励、特殊奖励等
//   totalWin: 500, // 本次总奖励
//   bet: 100, // 本次下注
//   freeSpinsRemaining: 15, // 剩余免费旋转次数
//   multiplier: 2, // 当前奖励乘数
//   timestamp: 1234567890
// }
```

## 后端对接说明

### API工具类 SlotApi
已封装好后端对接接口，包含：
- `validateSpin()`: 后端验证旋转结果（防作弊）
- `syncReward()`: 同步奖励到用户账户
- `triggerMiniGame()`: 触发联动小游戏
- `getUserData()`: 获取用户Slots相关数据
- `saveUserSettings()`: 保存用户设置

### 对接流程
1. 用户登录后调用 `SlotApi.init(userId, sessionToken)` 初始化
2. 旋转前可选择调用 `validateSpin()` 验证结果合法性
3. 奖励发放后调用 `syncReward()` 同步到用户账户
4. 触发小游戏时调用 `triggerMiniGame()` 获取启动参数
